# About Vorici Calculator

Vorici Calculator is a focused Path of Exile tool built to help players estimate socket color odds and compare coloring strategies in one place.

The site is intentionally narrow in scope. It is designed around one main search intent: players who want to know the cheapest way to hit a target socket color setup on a specific item base.

## What the site covers

- socket color probability estimates
- Chromatic Orb cost expectations
- bench-assisted coloring strategy comparisons
- practical explanations for off-color crafting decisions

## Scope

The current version focuses on the English calculator experience and a small set of support pages. Additional guides or tools may be added later if they support the same Path of Exile coloring workflow.
