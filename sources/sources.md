拆解 Vorici Calculator 的算法。

我给你把 Vorici Calculator 的核心算法完整拆开。
其实它不是一个复杂 AI 算法，本质是 概率 + 组合枚举 + 成本期望值计算。

做一个网站只需要实现 三层逻辑：

1️⃣ Socket颜色概率模型
2️⃣ 指定颜色组合概率
3️⃣ 不同 crafting 方法的平均成本

下面是完整拆解。

一、Socket颜色概率模型（核心）

在 Path of Exile 里，每个 socket 的颜色概率由装备的 属性需求决定：

属性	对应颜色
STR	Red
DEX	Green
INT	Blue

权重公式：

Weight_red = STR + 10
Weight_green = DEX + 10
Weight_blue = INT + 10

然后计算概率：

P_red = Weight_red / (Weight_red + Weight_green + Weight_blue)
P_green = Weight_green / total
P_blue = Weight_blue / total
示例

装备需求：

STR = 138
DEX = 0
INT = 0

权重：

Red = 148
Green = 10
Blue = 10

概率：

Red ≈ 88%
Green ≈ 6%
Blue ≈ 6%

这就是为什么 STR 装备几乎只出红孔。

二、多 socket 颜色组合概率

假设装备有：

N sockets

每个 socket 是独立概率。

如果目标是：

R red
G green
B blue

满足：

R + G + B = N

概率公式：

P = N! / (R! G! B!) * (P_red^R) * (P_green^G) * (P_blue^B)

这是 多项分布 (multinomial distribution)。

示例

4 sockets
目标：

3 blue
1 green

概率：

P = 4! / (3!1!) * (P_blue^3) * (P_green^1)
三、Chromatic Orb 平均成本

Chromatic Orb 每次 重新随机生成全部 socket 颜色。

成功概率：

P_success

平均尝试次数：

Attempts = 1 / P_success

如果：

Chromatic cost = 1 per attempt

平均成本：

Cost = Attempts
示例

如果成功概率：

0.001

平均需要：

1000 chromatics

八、JS核心代码示例

简化版概率计算：

function socketProb(str, dex, int) {
    const wR = str + 10
    const wG = dex + 10
    const wB = int + 10

    const total = wR + wG + wB

    return {
        red: wR / total,
        green: wG / total,
        blue: wB / total
    }
}

组合概率：

function factorial(n){
  if(n<=1) return 1
  return n*factorial(n-1)
}

function multinomial(n,r,g,b,pR,pG,pB){
  return factorial(n)/(factorial(r)*factorial(g)*factorial(b))*
         Math.pow(pR,r)*
         Math.pow(pG,g)*
         Math.pow(pB,b)
}

What is a Vorici Calculator?

A Vorici Calculator helps Path of Exile players estimate the cheapest way to get specific socket colors on an item.

How do socket colors work in Path of Exile?

Socket colors are weighted by the item’s attribute requirements. Strength favors red, Dexterity favors green, and Intelligence favors blue.

Is bench crafting always cheaper?

No. Sometimes Chromatic Orb spam is cheaper. The best method depends on the item and target colors.

Does this calculator work for 1 to 6 sockets?

Yes. It supports all common socket counts.

What is the Jeweller’s Method?

It is a technique used to roll off-colors by changing socket counts while preserving useful colors.
